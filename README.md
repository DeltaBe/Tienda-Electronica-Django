# 🛒 TechStore - E-commerce Full-Stack con Django

¡Bienvenido a **TechStore**! Esta es una aplicación web completa diseñada para la gestión de inventarios y ventas, desarrollada con un enfoque en la robustez de datos y la escalabilidad.

---

## 🚀 Demo en Vivo
Puedes ver la aplicación funcionando en tiempo real aquí:
👉 **[TechStore en Render](https://tienda-electronica-django.onrender.com)**  
*(Nota: Debido al plan gratuito de Render, la aplicación puede tardar unos 30-60 segundos en iniciar tras un periodo de inactividad).*

---

## 🛠️ Stack Tecnológico
*   **Backend:** [Django 5.x](https://djangoproject.com) (Python)
*   **Base de Datos:** [PostgreSQL](https://postgresql.org)
*   **Frontend:** HTML5, CSS3 (Bootstrap)
*   **Despliegue:** [Render](https://render.com)

---

## ✨ Características Principales
*   **Gestión de Inventarios:** CRUD completo para productos y categorías.
*   **Base de Datos Relacional:** Implementación de PostgreSQL para asegurar la integridad de los datos de ventas y usuarios.
*   **Autenticación de Usuarios:** Registro, inicio de sesión y gestión de perfiles.
*   **Diseño Responsivo:** Adaptado para dispositivos móviles y escritorio.
*   **Panel de Administración:** Configuración personalizada del Admin de Django para control interno.
